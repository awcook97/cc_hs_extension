Privacy Policy

Extension: HubSpot CompanyCam Viewer

Effective date: February 14, 2026

1. Overview

HubSpot CompanyCam Viewer ("the extension") helps users preview CompanyCam images referenced in HubSpot pages. The extension is designed to process data locally in the browser for this single purpose.

2. Data the extension accesses

The extension may access the following data in order to work:

  • Website content on https://*.hubspot.com/*, specifically text in the configured photo list field to detect CompanyCam image URLs.
  • Image URLs and image content from https://img.companycam.com/* to render previews and thumbnails.
  • User settings saved in browser extension storage (storage.sync), such as preview mode, sizes, and hover behavior.

3. What the extension does not do

  • Does not sell personal data.
  • Does not use data for advertising.
  • Does not use data for creditworthiness or lending decisions.
  • Does not run analytics, telemetry, or tracking beacons to developer-controlled services.
  • Does not collect passwords, payment data, health data, or precise location data.

4. Data sharing

The extension does not transfer user data to third parties except what is required to load HubSpot pages and CompanyCam-hosted images in normal browser operation. No separate developer backend is used for data collection or profiling.

5. Data retention

The extension stores only configuration settings in browser extension storage. These settings remain until the user changes them, clears extension data, or uninstalls the extension.

6. Security

The extension ships packaged code and uses browser extension APIs. While reasonable efforts are made to keep the extension secure, no method of software security is guaranteed to be perfect.

7. Changes to this policy

This policy may be updated from time to time. Material updates will be reflected by changing the effective date on this page.
